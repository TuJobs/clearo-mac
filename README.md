# Clearo

🇻🇳 **Tiếng Việt**  ·  [🇬🇧 English](README.en.md)

Ứng dụng dọn dẹp macOS gọn nhẹ, an toàn — **miễn phí**. Giao diện song ngữ Việt / English.

**[⬇️ Tải bản mới nhất (.dmg)](https://github.com/TuJobs/clearo-mac/releases/latest)**

> Yêu cầu: **macOS 11+ · Apple Silicon (M1+)** — *chưa hỗ trợ Mac Intel*

> ⚠️ **Miễn trừ trách nhiệm** — App **xoá tệp**, cung cấp **"nguyên trạng", KHÔNG bảo hành**; tác giả **không chịu trách nhiệm** cho mất mát dữ liệu. **Hãy sao lưu dữ liệu quan trọng trước khi dùng.** *(chi tiết ở [cuối trang](#-miễn-trừ-trách-nhiệm))*

Dọn cache & file rác, gỡ ứng dụng kèm file thừa, tìm file lớn/cũ & trùng lặp, xoá dấu vết riêng tư, và chạy các tác vụ bảo trì chuẩn của macOS. Mặc định mọi thứ chuyển vào **Thùng rác** (khôi phục được).

### Tính năng

- **Dọn nhanh** — một chạm quét & dọn cache, log, rác lập trình
- **Gỡ ứng dụng** — xoá app kèm toàn bộ file thừa còn sót
- **File lớn & cũ · Quét sâu · Trùng lặp** — tìm chỗ ngốn ổ đĩa
- **Riêng tư** — xoá cache / lịch sử / cookie trình duyệt
- **Bảo trì** — các tác vụ housekeeping chuẩn của macOS

### An toàn

- Mặc định mọi thứ vào **Thùng rác** (khôi phục được)
- Hiện rõ **mức độ an toàn 🟢/🟡/🔴** từng nhóm — tránh xoá nhầm

### Cài đặt

1. Tải `Clearo-x.y.z.dmg` ở mục **Releases** → mở → kéo **Clearo** vào **Applications**.
2. Lần đầu mở, macOS hỏi xác nhận → bấm **Open**.
3. Khi quét, cấp **Full Disk Access** nếu được hỏi (app có hướng dẫn sẵn).

> App đã **ký Developer ID + notarize bởi Apple** — cài không bị Gatekeeper chặn.
