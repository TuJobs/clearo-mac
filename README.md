# Clearo

Ứng dụng dọn dẹp macOS gọn nhẹ, an toàn — **miễn phí**. Giao diện **song ngữ Việt / English**.

Dọn cache & file rác, gỡ ứng dụng kèm file thừa, tìm file lớn/cũ & trùng lặp, xoá dấu vết
riêng tư, và chạy các tác vụ bảo trì chuẩn của macOS. Mặc định mọi thứ chuyển vào **Thùng rác**
(khôi phục được).

## ⬇️ Tải về

**[Tải bản mới nhất (.dmg)](https://github.com/TuJobs/clearo-mac/releases/latest)**

## Yêu cầu

- macOS trên **chip Apple Silicon** (M1, M2, M3, …) — *chưa hỗ trợ Mac dùng chip Intel*

## Cài đặt

1. Tải file `Clearo-x.y.z.dmg` ở mục **Releases**.
2. Mở file `.dmg`, kéo **Clearo** vào thư mục **Applications**.
3. Lần đầu mở, macOS hỏi xác nhận (vì tải từ Internet) → bấm **Open**.
4. Khi quét, cấp **Full Disk Access** nếu được hỏi (app có hướng dẫn sẵn trong giao diện).

> App đã được **ký Developer ID + notarize bởi Apple** nên **không bị Gatekeeper chặn** —
> không có cảnh báo "unidentified developer" hay "damaged".

## Tính năng

| | |
