# Clearo

[🇻🇳 Tiếng Việt](README.md)  ·  🇬🇧 **English**

A lightweight, safe macOS cleanup app — **free**. Bilingual Vietnamese / English UI.

**[⬇️ Download the latest release (.dmg)](https://github.com/TuJobs/clearo-mac/releases/latest)**

> Requires: **macOS 11+ · Apple Silicon (M1+)** — *no Intel support yet*

> ⚠️ **Disclaimer** — This app **deletes files**, provided **"AS IS" with NO WARRANTY**; the author is **not liable** for data loss. **Back up important data before use.** *(full text at the [bottom](#-disclaimer))*

Clean caches & junk, uninstall apps with their leftovers, find large/old & duplicate files, clear browser traces, and run standard macOS maintenance. Everything goes to the **Trash** (recoverable) by default.

### Features

- **Quick Clean** — one tap to scan & clean caches, logs, dev junk
- **Uninstaller** — remove apps along with all their leftover files
- **Large & Old Files · Deep Scan · Duplicates** — find what's eating your disk
- **Privacy** — clear browser cache / history / cookies
- **Maintenance** — standard macOS housekeeping tasks

### Safety

- Everything goes to the **Trash** (recoverable) by default
- Each group shows a **🟢/🟡/🔴 risk level** so you don't delete something you need

### Install

1. Download `Clearo-x.y.z.dmg` from **Releases** → open → drag **Clearo** to **Applications**.
2. On first launch, macOS asks to confirm → click **Open**.
3. Grant **Full Disk Access** if prompted while scanning (the app guides you).

> Developer-ID signed + **notarized by Apple** — installs with no Gatekeeper warnings.
